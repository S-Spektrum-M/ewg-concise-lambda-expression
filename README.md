# Concise Lambda Expressions (WG21 Proposal)

This repository contains the source code, tooling, and drafts for the C++ Evolution Working Group (EWG) proposal
to introduce a concise lambda expression syntax to C++.

* Document Number: PXXXXR0
* Title: Concise Lambda Expressions
* Author: Siddharth Mohanty

## Proposal Draft

Whenever changes are pushed to the `main` branch, the proposal is automatically compiled and published:
* Read Online (HTML) [https://S-Spektrum-M.github.io/ewg-concise-lambda-expression/](https://S-Spektrum-M.github.io/ewg-concise-lambda-expression/)
* Download Draft (PDF) [https://S-Spektrum-M.github.io/ewg-concise-lambda-expression/PROPOSAL.pdf](https://S-Spektrum-M.github.io/ewg-concise-lambda-expression/PROPOSAL.pdf)
